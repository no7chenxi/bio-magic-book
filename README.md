# 🧪 Genos Bio-AI Magic Codex (身体魔法书)

> **"Every cell is a line of code. Every heartbeat is an instruction."**

这是一个专为 **1-10 岁** 小朋友设计的沉浸式生命科学与 AI 启蒙交互网页应用。通过充满童趣的交互和优美的音乐，带领孩子破译身体里的 22 个魔法秘密。

## 🌟 项目亮点
- **跨界启蒙**：将生物学（DNA、器官）与计算机科学（代码、CPU、电源管理）一一对齐，培养孩子的系统性思维。
- **全感官互动**：采用 **3 秒蓄力解锁** 逻辑，配合 **Web Audio API** 实时合成的钢琴音效与触感反馈。
- **艺术氛围**：背景音乐采用 Joe Hisaishi 经典曲目 **《 Summer 》**，营造清新、愉悦的夏日探险感。
- **博物馆收集**：内建 4 大模块收集地图（感知、动力、系统、宇宙），激发孩子的收集成就感。
- **Genos 赋能**：科普 Genos 基因组大模型的 **1Mb 超长上下文** 处理能力，将其具象化为“远端控制开关”。
- **全平台适配**：针对手机端、iPad 和 PC 进行深度布局优化，确保“零溢出”的完美视觉体验。

## 🎮 玩法介绍
1. **启动**：点击“启动魔法之旅”唤醒背景音乐。
2. **寻找**：移动“手电筒”光圈在黑暗的微观宇宙中寻找发光的秘密图标。
3. **解码**：将光圈圆心对准图标并**稳住 3 秒钟**，看着进度环合拢。
4. **存入**：读完关于超能力的秘密解释，看着它飞入底部的博物馆格子。

## 🔬 技术栈
- **Frontend**: HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript.
- **Audio**: Web Audio API (实时正弦波钢琴音色合成)。
- **Logic**: 基于网格的响应式随机分布算法。
