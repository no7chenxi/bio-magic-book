# 🧪 Genos Bio-AI Magic Codex (身体魔法书)

> **"Every cell is a line of code. Every heartbeat is an instruction."**

这是一个专为 **1-10 岁** 小朋友设计的沉浸式生命科学与 AI 启蒙交互页面。

## 🌟 项目亮点
- **跨界启蒙**：将生物学（DNA、器官）与计算机科学（代码、CPU、杀毒软件）一一对齐，培养孩子的系统性思维。
- **全感官互动**：采用 3 秒蓄力解锁逻辑，配合实时合成的 Web Audio 音效与触感反馈。
- **Genos 赋能**：通过简单的“魔法放大镜”比喻，科普 Genos 基因组大模型的 1Mb 超长上下文处理能力。
- **全平台适配**：基于动态网格分布算法，确保在任何屏幕（手机、iPad、PC）上都能完美展现 22 个生命秘密。

## 🎮 如何开始
1. 运行 `index.html` 即可开启魔法探索。
2. 移动“手电筒”光圈寻找隐藏的秘密。
3. 对准圆心并保持 3 秒，听着能量积聚的声音，点亮你的生命博物馆！

## 🔬 技术背景
本项目是基于 **BGI-Research & 浙江实验室** 发布的 **Genos 基因组大模型** 技术理念开发的科普教育演示。通过模拟微观世界的探索，让孩子从小理解“远端调控”和“生命代码”的神奇。
