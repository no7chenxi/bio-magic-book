# 🌌 Genos Bio-AI Galaxy Codex (星际生命魔法书)

> **"Exploring the infinite universe within every cell."**

这是一个专为 **1-10 岁** 小朋友设计的、具有电影级视觉冲击力的沉浸式生命科学与 AI 启蒙交互应用。将生命微观世界转化为浩瀚宇宙，带领孩子在星辰大海中破译身体的代码。

## 🌟 终极版本亮点
- **浩瀚星际视觉**：采用动态生成的星空背景、闪烁繁星与划过天际的流星特效，营造极致的探索氛围。
- **巨大化图标聚焦**：扫描仪圆心对准秘密时，图标会瞬间膨胀 2.2 倍并全彩发亮，带来极强的“镜头拉近”爽感。
- **跨界硬核启蒙**：将生物器官（大脑、心脏、神经）与 AI 组件（CPU、电源管理、总线）深度对齐，用童话语境传递前沿科学。
- **全感官沉浸**：集成 Joe Hisaishi 的经典曲目 **《 Summer 》** 钢琴合成音，配合气泡破裂、弹簧拉伸等趣味打击音效。
- **全机型零溢出适配**：基于动态网格与安全区算法，确保在任何手机（iOS/安卓）及平板上都能完美呈现，按钮无遮挡。
- **大师颁奖礼**：集齐 22 个秘密后触发终极星图合成仪式，颁发“生命代码大师”荣誉证书。

## 🎮 玩法介绍
1. **启航**：点击“开启星际探险”激活宇宙音效。
2. **扫描**：移动光圈准星在繁星中寻找巨大的生命星球图标。
3. **锁定**：将圆心稳住对准图标 **3 秒钟**，看着能量圆环合拢。
4. **归档**：读完关于超能力的跨界解释，看它划出优美弧线飞入底部的荣誉博物馆。
