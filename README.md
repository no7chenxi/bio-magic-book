# 🌌 Genos Bio-AI Galaxy Codex (星际生命魔法书)

> **"Exploring the infinite universe within every cell."**

这是一个专为 **1-10 岁** 小朋友设计的、具有电影级视觉冲击力的沉浸式生命科学与 AI 启蒙交互应用。将生命微观世界转化为浩瀚宇宙，带领孩子在星辰大海中破译身体的代码。

## 🌟 终极版本亮点
- **浩瀚星际视觉**：采用动态生成的星空背景、闪烁繁星与划过天际的流星特效，营造极致的探索氛围。
- **巨大化图标聚焦**：扫描仪圆心对准秘密时，图标会瞬间膨胀 2.2 倍并全彩发亮，带来极强的“镜头拉近”爽感。
- **跨界硬核启蒙**：将生物器官（大脑、心脏、神经）与 AI 组件（CPU、电源管理、总线）深度对齐，用童话语境传递前沿科学。
- **全感官沉浸**：集成 Joe Hisaishi 的经典曲目 **《 Summer 》** 钢琴合成音，配合气泡破裂、弹簧拉伸等趣味打击音效。
- **全机型零溢出适配**：基于动态网格与安全区算法，确保在任何手机（iOS/安卓）及平板上都能完美呈现，按钮无遮挡。
- **大师颁奖礼**：集齐 22 个秘密后触发终极星图合成仪式，颁发“生命代码大师”荣誉证书。

## 📖 背后的故事 (Behind the Scenes)

### 💡 项目启发
本项目始于对 **Genos 基因组大模型** 的深度探索。在思考如何向 1-10 岁的孩子解释“100 亿参数”和“1Mb 超长上下文”时，我们发现生命的调控逻辑（DNA -> 蛋白 -> 功能）与人工智能的底层逻辑（代码 -> 模型 -> 执行）有着惊人的相似性。于是，我们决定打造一个“跨界翻译官”，让孩子在玩耍中同时开启生物学与计算机科学的兴趣之门。

### 🧬 设计进化历程
- **阶段 1：实验室黑客风** —— 尝试用最严谨的术语对齐，但发现对孩子来说过于冰冷。
- **阶段 2：萌宠养成模式** —— 引入了“糯米团子”形象，趣味性大增，但承载的海量知识库显得过于拥挤。
- **阶段 3：星际漫游终极版** —— 我们将微观世界升华为浩瀚宇宙，通过“准星锁定”和“3秒蓄力”模拟科学家破译代码的仪式感，最终实现了审美、趣味与知识深度的平衡。

### 🛠️ 技术挑战与破局
- **交互痛点**：在手机端，传统的 `hover` 逻辑失效，且光圈遮罩会拦截点击事件。
  - *解决*：重构了基于圆心距离检测的 `touchmove` 引擎，实现了 80px 的超大感应热区，确保“指哪打哪”。
- **信息过载**：22 个图标同时出现时，卡片极易遮挡目标。
  - *解决*：首创了“图标向上浮动 + 卡片向下弹出”的非遮挡布局，并引入了“智能安全区分布算法”，确保图标永不溢出屏幕。
- **沉浸感难题**：不依赖任何 mp3 外部文件，如何实现欢快音效？
  - *解决*：利用原生 **Web Audio API** 实时合成正弦波与正切波，模拟出《 Summer 》的钢琴质感与“Pop!”的气泡破裂声。

## 🔬 技术栈
- **Vfx**: 纯 CSS3 关键帧动画 + 动态随机星空生成。
- **Audio**: Web Audio API (正弦波钢琴音色 + 趣味 SFX 实时合成)。
- **Layout**: 移动端响应式“绝对安全区”分布算法。
